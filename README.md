# NetworkPlus

Network+ is a free, lightweight, and handy network tool for Windows that provides a collection of useful networking utilities. It includes a DNS changer, LAN scanner, ping tools, built-in DNS server lists for gamers and IT engineers, and more to help troubleshoot, analyze, and optimize networks.

Changelog:
✨ Refreshed with a new design and new branding.
⚡ Improved speed, performance, and overall stability.
🛠️ Fixed various known issues and bugs.
🌐 Added LAN Scan and support for managing custom DNS servers.
📋 Introduced categorized DNS lists: Global, Europe, Iran, and China.

## System Requirements

**Framework:** .NET Framework 4.5 (XP to Windows 11). Tested on Windows 10 and 7.

I made the first release of this project in late 2020 and then almost forgot about it. Later I decided to host it on GitHub.
